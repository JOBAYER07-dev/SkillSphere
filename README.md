🎓 SkillSphere -->>>
An online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs like Web Development, Design, Marketing, and more.
------------------------------------------------------------------
🌐 Live URL
🔗 https://skill-sphere-sable.vercel.app/
------------------------------------------------------------------
📌 Purpose-->>>
SkillSphere is a modern online learning platform built to help learners upgrade their skills through expert-led courses. Users can browse courses, view detailed curriculum, and enroll after logging in. The platform supports both email/password and Google authentication.
------------------------------------------------------------------
✨ Key Features-->>>
🏠 Home Page — Hero banner with slider, Popular Courses, Trending Now, Learning Tips, and Top Instructors sections
📚 All Courses Page — Browse all courses with search by title and filter by category
🔒 Protected Course Details — Course details page is only accessible to logged-in users
🔐 Authentication — Email/password login & registration, Google OAuth via BetterAuth
👤 User Profile — View logged-in user info with enrolled course stats
✏️ Update Profile — Update display name and profile photo
🔔 Toast Notifications — Success and error feedback on all actions
📱 Fully Responsive — Works on mobile, tablet, and desktop
❌ 404 Not Found Page — Custom page for invalid routes
⚡ Smooth Animations — Motion animations on sections and cards
